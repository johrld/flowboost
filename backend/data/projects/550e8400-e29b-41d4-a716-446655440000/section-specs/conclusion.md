# Section Spec: Conclusion (Fazit + CTA)

## Ziel

Schließe den Artikel mit einer motivierenden Zusammenfassung und einem klaren Call-to-Action ab.

## Struktur

```markdown
## Fazit

[Absatz 1: Zusammenfassung der 3 Key Takeaways - 3-4 Sätze]

[Absatz 2: Ermutigung + CTA - 2-3 Sätze]
```

## Anforderungen

| Eigenschaft | Wert |
|-------------|------|
| Gesamtlänge | 100-200 Wörter |
| Absätze | 2-3 |
| Key Takeaways | 3 Hauptpunkte zusammenfassen |
| CTA | 1x App-Download oder verwandter Artikel |
| Primary Keyword | Einmal natürlich einbauen |

## Inhalt

### Absatz 1: Zusammenfassung

- Fasse die 3 wichtigsten Erkenntnisse zusammen
- Beziehe dich auf die Key Points aus der Outline
- Keine neuen Informationen einführen
- Kompakt und klar formulieren

### Absatz 2: Ermutigung + CTA

- Ermutige den Leser zum Ausprobieren
- Breathe-typischer Ton: warm, nicht preachy
- CTA-Typen (basierend auf `cta_type` aus Outline):
  - `app_download`: "Probiere es aus mit der kostenlosen Breathe App"
  - `related_article`: "Erfahre mehr in unserem Guide zu [Thema]"
  - `practice`: "Starte heute mit deiner ersten Session"

## CTA Format

```markdown
Starte jetzt mit [Technik] – die [Breathe App](/de/app) begleitet dich Schritt für Schritt mit geführten Sessions.
```

## Verboten

- Neue Konzepte oder Informationen einführen
- "Zusammenfassend..." oder "Abschließend..." als Opener
- Übertriebene Versprechen ("wird dein Leben verändern")
- Mehr als einen CTA

## Output-Datei

Speichere als `conclusion.md` im Scratchpad. Beginne mit `## Fazit`.
