# Article Writing Skill

## Structure

Every article follows this structure:

1. **H1 Title** — contains primary keyword, 50-70 characters
2. **Answer Capsule** — blockquote after H1, 2-3 sentences answering the core question
3. **Introduction** — hook + context, 100-200 words
4. **H2 Sections** (3-5) — main content, 200-400 words each
5. **Conclusion** — summary + CTA, 100-200 words
6. **FAQ** — 3-5 questions in frontmatter (rendered as accordion)

## SEO Rules

- Primary keyword in H1, first 100 words, and at least 2 H2s
- Keyword density: 1-2% (natural, never forced)
- Internal links: minimum 2 per article
- Meta description: 100-160 characters with keyword + benefit
- URL slug: short, keyword-rich, no stop words

## Writing Style

- Short sentences (max 20 words average)
- Short paragraphs (max 4 sentences)
- Active voice preferred
- Concrete examples over abstract claims
- Numbers and data where possible

## Answer Capsule Format

```markdown
> **Short Answer:** [Direct answer in 2-3 sentences]
> [One supporting fact or next step]
```

This appears directly after the H1. Optimized for Featured Snippets and AI search citations.

## FAQ Rules

- Questions as users would ask them (natural language)
- Answers: 40-60 words each
- Include primary keyword in at least one question
- No promotional questions ("Why should I use X?")

## Do

- Start sections with the key point, then elaborate
- Use transition sentences between H2 sections
- Reference sources and data
- Write for scan-readers (headings, bold, bullets)

## Don't

- Use "In this article, we will..." introductions
- Write sections longer than 500 words
- Use H3 without a parent H2
- Add FAQ section in the content body (it goes in frontmatter)
