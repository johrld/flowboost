# Newsletter Skill

## Subject Line

- 40-60 characters (shows fully on mobile)
- Create curiosity or promise specific value
- No clickbait — deliver on the promise
- Personalization when possible
- Test: "Would I open this?"

## Preview Text

- 80-120 characters
- Complements the subject (doesn't repeat it)
- Adds context that makes the subject more compelling

## Structure

1. **Greeting** — brief, personal
2. **Introduction** (2-3 sentences) — what's in this issue and why it matters
3. **Main Content** (2-3 sections) — the value
4. **CTA** — one clear action
5. **Sign-off** — personal, brief

## Section Guidelines

- Each section: heading + 2-4 paragraphs
- Short paragraphs (2-3 sentences)
- One key takeaway per section
- Use bullet points for lists
- Include at least one link per section

## Tone

- Personal and direct — write to one person
- Conversational but substantive
- Like a smart friend sharing what they learned
- Avoid marketing-speak

## Do

- Start with the most valuable content
- Use clear section headings
- Include one specific, actionable CTA
- Keep total length 300-800 words

## Don't

- Include more than one primary CTA
- Write walls of text without headings
- Use "Dear subscriber" or generic greetings
- Send without proofreading subject + preview
